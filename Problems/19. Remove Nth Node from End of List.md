## Ideas
1. L,R
2. R=L+n (for循环)
3. while R!= null:  R+=1, L+=1
4. remove(L)
5. return
```
class Soluton(object):
  def removeNthFromEnd(self, head, n):
    dummy = ListNode(0, head)
    left=dummy
    right=dummy.next

    for i in range(n):
      right=right.next

    while right:
      right=right.next
      left=left.next

    # delete the number
    left.next=left.next.next
    return dummy.next
```
