## HashMap
- HashMap可以是指一个dict{}对象，也可以是指一个list[]对象
- 本题中就是list对象
```
class Solution(object):
    def isIsomorphic(self, s, t):
        """
        :type s: str
        :type t: str
        :rtype: bool
        """
        slist=[]
        tlist=[]

        for char in s:
            slist.append(s.index(char))
        for char in t:
            tlist.append(t.index(char))

        if slist == tlist:
            return True
        else:
            return False
```
