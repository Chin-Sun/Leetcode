## Greedy----O(n) space
```
class Solution(object):
    def canJump(self, nums):
        """
        :type nums: List[int]
        :rtype: bool
        """
        goal = len(nums)-1

        for i in range(len(nums) -1,-1,-1):
            if i+nums[i] >= goal:
                goal = i
        return True if goal == 0 else False
```
- range(len(nums)-1, -1, -1)ï¼Œ the first "-1" means the loop stops at the -1+1=0th position. So although it sets "-1", it never arrives "-1".
